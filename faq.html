FAQs

  • How to place a new Order?

    You put the service ID followed by | followed by the link followed by | followed by quantity on each line
    To get the service ID of a service please check here:https://follow.sale/services
    Let's say you want to use the Mass Order to add Instagram Followers to your 3 accounts: abcd, asdf, qwer
    From the Services List @ https://follow.sale/services, the service ID for this service "Instagram Followers [15K] [REAL]" is 33
    Let's say you want to add 1000 followers for each account, the output will be like this:
    ID|Link|Quantity
    or in this example:
    102|abcd|1000
    102|asdf|1000
    102|qwer|1000
